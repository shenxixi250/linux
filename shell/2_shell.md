##  程序语言执行

| 语言   | 执行方式                          |
|--------|-----------------------------------|
| C      | ====编译====二进制机器码(cpu)     |
| java   | ==编译==字节码(java jvm)          |
| shell  | ==解释                            |
| python | 解释或者编译(字节码 python虚拟机) |

编译
  + 快  但是如果我要改一个地方就要重新编译不方便

解释 
  + 就是慢    但是比较方便
